Percentage Change Distribution(Hover Me) The histogram below shows the distribution of percentage change numbers from base time period x to non-base time period(s). Use the slider to filter countries in the table that had a percentage increase of more than x(where x is the number on the slider and input box) from base time period x to non base time period y.
Absolute Change Distribution(Hover Me) The histogram below shows the distribution of absolute change numbers from base time period x to non-base time period(s). Use the slider to filter countries in the table that had an absolute increase of more than x(where x is the number on the slider and input box) from base time period x to non base time period y.
Anomaly Percentage(Hover Me) The histogram above shows the distribution of percentage change numbers from any time period x to any time period y. Unlike the percentage change distribution slider, the distribution shown in this slider does not depend on the current base time period. Use the slider to change the threshold for percentage anomalies.
Anomaly Absolute(Hover Me) The histogram above shows the distribution of absolute numbers for all the time periods. Use the slider to change the threshold for absolute anomalies.
